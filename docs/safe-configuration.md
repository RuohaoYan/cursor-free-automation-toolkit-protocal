# 安全配置默认值说明

本项目的示例配置用于说明可配置项，不应携带真实账号、密钥、手机号、代理凭据或卡密接口信息。

## 本次调整

- 示例代理服务器、用户名、密码、session 改为占位值。
- 示例手机号、支付 PIN、ADB 设备号保持为空。
- 短信、外部代理、自动兑换、自动接码、协议注册等外部服务开关默认关闭。
- 本地授权服务默认绑定到 `127.0.0.1`，避免示例配置暴露到公网。
- 需要调用外部服务的配置项保留字段，但必须由使用者在确认授权、合规和测试环境后显式启用。

## 使用建议

1. 从 `.env.example` 复制出 `.env` 后，只填写当前要验证的最小流程所需字段。
2. 不要把 `.env`、账号池、手机号池、卡池、代理池、token、session 或输出成品提交到仓库。
3. 修改外部服务开关前，先在单账号、单线程、测试环境中验证配置是否完整。
4. 发现项目无法启动时，优先检查依赖安装、配置文件缺失、路径不存在和必填字段为空的问题。

## 验证边界

本次变更只调整示例配置和说明文档，不修改注册、支付、验证码、授权、会话导出等业务流程实现。
